                                /*LOGIC FUNCTIONS*/
                                //reverse input string
                                function reverseString (str) 
                                {
                                    return str.split('').reduce((reversed, character) => character + reversed, '')
                                }

                                /* DISPLAY OR VIEW FUNCTIONS*/
                                //display the reversed string
                                function displayString(revStr, isReversible = true)
                                {
                                    //clear styling on alert box
                                    document.getElementById("alert").classList.remove("alert-success", "alert-danger", "alert-warning", "alert-info");

                                    if(isReversible)
                                    {
                                        //write to page
                                        document.getElementById("alertHeading").innerHTML = `Success!`;
                                        document.getElementById("msg").innerHTML = `Your string reversed is:${revStr}`;
                                        //style alert box
                                        document.getElementById("alert").classList.add("alert-success");
                                    }
                                    else
                                    {
                                        //write to page
                                        document.getElementById("alertHeading").innerHTML = `Error`;
                                        document.getElementById("msg").innerHTML = `Nothing to reverse! Please enter a string.`;
                                        //style alert box
                                        document.getElementById("alert").classList.add("alert-danger");
                                    }
                                    
                                    //show the alert box
                                    document.getElementById("alert").classList.remove("invisible");
                                    document.getElementById("alert").classList.add("show");

The code is structured with 3 functions.

getInput()

This function is responsible for controlling the flow of user interaction with the page.

  • It sets the visibility of the alert element with the ID "alert" to invisible.
  • It then collects user input from an element with the ID "inputStr".
  • The user input is validated by checking if it is empty, in which it calls the displayString() function with the boolean IsReversible parameter set to false, indicating that there's nothing to reverse.
  • If there is valid input, the function reverses the string using the reverseString() function and displays the reversed result through the displayString() function.

reverseString(str)

This function is responsible for reversing the input string.

  • It uses the split method to turn the input string str into an array of characters, and then it uses the reduce method to reverse the order of these characters.
  • This is achieved by concatenating each character to the beginning of the reversed string.
  • The function returns the reversed string.

displayString(revStr, isReversible = true)

This function handles the display of results and alert messages.

  • It starts by clearing any styling on the alert box by removing classes like alert-success, alert-danger, alert-warning, and alert-info.
  • If the isReversible flag is true (by default), it displays a success message. It sets the alert box heading to "Success!" and displays the reversed string in the alert message area. It also styles the alert box as a success alert.
  • If the isReversible flag is false, it displays an error message. It sets the alert box heading to "Error" and provides an error message indicating that there's "Nothing to reverse!" It styles the alert box as an error with the alert-danger style.
  • It then makes the alert box visible by removing the "invisible" class and adding the "show" class.
